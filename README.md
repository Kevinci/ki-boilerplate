# ki-boilerplate
ITCSS Boilerplate

# Installation

Install SASS
```shell
gem install sass --pre
```

Install Gulp
```shell
npm install -g gulp
```
Install Grunt
```shell
npm install -g grunt-cli
```
## Usage

```shell
git clone git@github.com:evinci/ki-boilerplate.git
```
**Installing the dependencies of project**
```shell
npm install or yarn or yarn install 
```
